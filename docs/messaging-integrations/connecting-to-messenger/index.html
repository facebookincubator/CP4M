Connecting to Messenger

To connect CP4M to Facebook Messenger, you will need:

  1. A Facebook account registered as a Meta developer
  2. A Meta app to integrate with Meta APIs
  3. A Facebook Page for users to interact with over Messenger

Getting Meta developer access​

To register as a Meta developer, you can follow the steps here.

Creating a Meta app​

You can see all of your existing Meta developer apps in the Meta Developer App Dashboard.

To create a new app, click on the Create App button. The steps for the app creation process are detailed here.

Create a Facebook page​

If you don't already have a Facebook page for Messenger interactions, you can follow the steps here to create a new page.

Enabling Messenger API Access on your Meta app​

In the Meta Developer App Dashboard:

  1. Click on your Meta app
  2. On the Dashboard page, find the Add products to your app section.
  3. On the Messenger tile, click the Set up button.

  1. On the "1. Configure webhooks" section, click the Configure button.

  1. Follow the steps in the Messenger Webhooks documentation to configure the Callback URL and Verification Token for your server.

  1. On the "2. Generate access tokens" section, click the Add Subscriptions button on an existing Facebook page which you would like your app to have Messenger API access.

  1. At minimum, select the following Page Subscriptions. Learn more about all the Page Subscription options here: a. messages b. messaging_postbacks

  2. Click the Confirm button to save your page subscription choices.

  1. Finally, on the "3. Complete App Review" section, follow the steps to complete the App Review process so that your app can access production data. You can learn more about Meta's app review process here.
